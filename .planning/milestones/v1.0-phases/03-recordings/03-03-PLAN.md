    3. Clear `DisplayItems` and add all results
    4. Set `IsEmpty = DisplayItems.Count == 0`
    5. Set `HasActiveFilter = SelectedFilter is not null`

    **OnSelectedFilterChanged partial method:** Call `ApplyFilter()` when filter changes.

    **Subscribe to device connection state** in constructor: listen to `IDeviceService.ConnectionStateChanged`, update `IsDeviceConnected`, and reload if newly connected.
